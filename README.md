# portola

Transforms folders and images into a fast, modern portfolio website. Focus on content and styling instead of worrying about how it all works.

## directory structure

Here is an example of how a portola project wold look.

### complicated project (ambitious hope)

- content/
  - index.md
  - Blog/
    - First Blog Post/
      - index.md
    - Second Blog Post/
      - index.md
      - x.jpg
      - y.jpg
      - z.jpg
  - Photos/
    - Landscapes/
      - Big Sur.jpg
      - Tiburon.jpg
    - Portraits/
      - Joey.jpg
      - Maya.jpg
    - Street/
      - Berlin.jpg
      - San Francisco.jpg
- theme/
  - collections/
    - image.js
    - text.js
  - pages/
    - home.js
    - image.js
    - text.js
- static/
  - favicon.jpg
  - social.jpg
  - style.css
